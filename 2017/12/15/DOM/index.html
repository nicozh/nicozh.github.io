DOM Nodes

  1. 整个文档是一个文档节点
  2. 每个 HTML 元素是元素节点
  3. HTML 元素内的文本是文本节点
  4. 每个 HTML 属性是属性节点
  5. 注释是注释节点

parent、child和sibling

  1. 节点树中的节点彼此拥有层级关系。
  2. 在节点树中，顶端节点被称为根（root）。
  3. 每个节点都有父节点、除了根（它没有父节点）。
  4. 一个节点可拥有任意数量的子节点。
  5. 同胞是拥有相同父节点的节点。

Node 的接口

属性

nodeName,nodeType

nodeName属性返回节点的名称，nodeType属性返回节点类型的常数值。具体值查看MDN

Node.nodeValue

对于文档节点来说, nodeValue返回null. 对于text, comment, 和 CDATA 节点来说, nodeValue返回该节点的文本内容. 对于 attribute 节点来说, 返回该属性的属性值.

Node.textContent

Node.textContent属性自动忽略当前节点内部的HTML标签，返回所有文本内容。该属性是可读写的，设置该属性的值，会用一个新的文本节点，替换所有原来的子节点。它的一个好处就是自动对HTML标签转义。

对于Text节点和Comment节点，该属性的值与nodeValue属性相同。其他类型的节点，该属性会将每个子节点的内容连接在一起返回，但是不包括Comment节点。如果一个节点没有子节点，则返回空字符串。
document节点和doctype节点的textContent属性为null。读取整个文档的内容使用document.documentElement.textContent。

Node.nextSibling

Node.nextSibling属性返回紧跟在当前节点后面的第一个同级节点,是一个只读属性。如果当前节点后面没有同级节点，则返回null。
注意：该属性还包括文本节点和评论节点。因此如果当前节点后面有空格或者回车，该属性会返回一个文本节点，内容为空格或者回车。

Node.previousSibling

返回当前节点的前一个兄弟节点,没有则返回null.
如果当前节点后前面有空格或者回车，该属性会返回一个文本节点，内容为空格或者回车。

Node.ownerDocument

Node.ownerDocument属性返回当前节点所在的顶层文档对象，即document对象。document 对象是在实际的HTML文档中的所有子节点所属的主对象,document对象本身的ownerDocument属性，返回null。

Node.parentNode

parentNode属性返回当前节点的父节点。一个元素节点的父节点可能是一个元素(Element )节点,也可能是一个文档(Document )节点,或者是个文档碎片(DocumentFragment)节点.

Node.childNodes

childNodes属性返回一个伪数组NodeList集合，成员包括当前节点的所有子节点。注意，除了HTML元素节点，该属性返回的还包括Text节点(回车和空格也算)和Comment节点。如果当前节点不包括任何子节点，则返回一个空的NodeList集合。由于NodeList对象是一个动态集合，一旦子节点发生变化，立刻会反映在返回结果之中。

Node.appendChild()

Node.appendChild() 方法将一个节点添加到指定父节点的子节点列表末尾。

Node.cloneNode()

语法：

  1. node:将要被克隆的节点
  2. dupNode:克隆生成的副本节点
  3. deep: 可选是否采用深度克隆,如果为true,则该节点的所有后代节点也都会被克隆,如果为false,则只克隆该节点本身.(如果省略的话，参数的默认值为 false)

Node.removeChild()

Node.removeChild() 方法从DOM中删除一个子节点。返回删除的节点。
语法：

  1. child 是要移除的那个子节点.
  2. node 是child的父节点.
  3. oldChild保存对删除的子节点的引用. oldChild === child.

被移除的这个子节点仍然存在于内存中,只是没有添加到当前文档的DOM树中,因此,你还可以把这个节点重新添加回文档中，但你需要一个变量来保存这个节点的引用（oldChild）。

Node.replaceChild()方法用指定的节点替换当前节点的一个子节点，并返回被替换掉的节点。

  3. replacedNode 和oldChild相等。

Node.contains()

Node.contains()返回的是一个布尔值，来表示传入的节点是否为该节点的后代节点。
语法：

  1. node 是否包含otherNode节点.
  2. otherNode 是否是node的后代节点.
  3. 如果 otherNode 是 node 的后代节点或是 node 节点本身.则返回true , 否则返回 false.

Node.normalize()

normailize方法用于清理当前节点内部的所有Text节点。它会去除空的文本节点，并且将毗邻的文本节点合并成一个。

getElementById()

getElementById方法返回匹配指定id属性的元素节点。如果没有发现匹配的节点，则返回null。该方法的参数是大小写敏感的。比如，如果某个节点的id属性是main，那么document.getElementById(‘Main’)将返回null，而不是那个节点。

document.getElementById方法与document.querySelector方法都能获取元素节点，不同之处是document.querySelector方法的参数使用CSS选择器语法，document.getElementById方法的参数是HTML标签元素的id属性。

注：这个方法只能在document对象上使用，不能在其他元素节点上使用。

document.createElement()

document.createElement方法用来生成网页元素节点。参数为元素的标签名，是一个字符串。可大写，也可以小写，但不能包含尖括号”<>“

document.createTextNode()

创建一个新的文本节点，参数为一个字符串,包含了放在文本节点中的内容.

document.createAttribute()
